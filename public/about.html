ABOUT THE CURSED MINES GAME

Programmed by Iftah Haimovitch

Starting March 20 2013

A submission to Udacity Game Programming contest

Features including:

  • Isometric Map rendering and loading (Staggered style)
  • Isometric animated sprites
  • Random map generation - using two algorithms one on top of the other
  • Collision checking
  • Path finding
  • Particle system
  • Background processing (when generating Map)
  • Batch asset preloading
  • Sound & Music
  • Mini game of random math puzzles - with adjustable difficulty level
